第一章：日志查询（⭐） ​

1.1 概述 ​

  • Linux 中查询日志的时候，推荐使用占用系统资源少的命令，如：head、tail、less、more 命令。

注意⚠️：不推荐使用 vi、vim 或 cat 查询日志，如果使用这些命令，可能会造成系统卡死、内存不足，尤其在日志文件内容较大的情况下。

1.2 head 命令 ​

  • 命令：
shell
head [-n 数字] 文件
1

  • 功能：显示文件的前几行，默认是前 10 行。

  • 选项：

    • -n 数字，--lines 数字：指定要显示行的数量，-n 5 就表示前 5 行。

  • 示例：显示 /etc/passwd文件的前 10 行内容

shell
head /etc/passwd
1

  • 示例：显示/etc/passwd文件的前 5 行内容
shell
head -n 5 /etc/passwd
1

1.3 tail 命令 ​

  • 命令
shell
tail [-n 数字] [-f] 文件
1

  • 功能：显示文件的最后几行，默认是最后 10 行。

  • 选项：

    • -n 数字，--lines 数字：指定要显示行的数量，-n 5 就表示最后 5 行。
    • -f，--follow：显示文件末尾的实时更新，一般用于实时查看日志。

  • 示例：显示 /etc/passwd 的最后 10 行内容

shell
tail /etc/passwd
1

  • 示例：显示 /etc/passwd 的最后 5 行内容
shell
tail -n 5 /etc/passwd
1

  • 示例：实时查看文件末尾的更新
shell
tail -f /var/log/secure
1

1.4 less 命令 ​

  • 命令：
shell
less [-N] 文件
1
  • 功能：分屏上下翻页查看文件内容，即一页一页的查看文件内容。
  • 选项：
    • -N：每行显示行号。
  • 快捷键：
快捷键备注
q退出
y上一行
Enter下一行
空格下一页
b上一页，back 的缩写
g移动到首行
100g移动到第 100 行
G移动到最后一行
/搜索内容向下搜索，n (next)继续向下搜索，N 继续向上搜索。
  • 示例：显示 /etc/services 文件的内容
shell
less -N /etc/services
1

1.5 more 命令（了解） ​

  • 命令：
shell
more [-d][-c] 文件
1
  • 功能：more 命令和 less 命令类似，都是按页显示文件内容的；
  • 选项：
    • -d：显示"[press space to continue,'q' to quit.]"和"[Press 'h' for instructions]"。
    • -c：不进行滚屏操作。每次刷新这个屏幕。
  • 快捷键：
快捷键备注
q退出
Enter下一行
空格下一页
b上一页
h帮助

注意⚠️：

  • ① more 是一个基于 vi 编辑器的文本过滤器；所以实际中，针对日志查询，通常使用 less 命令。
  • ② more 命令到达最后一行就会退出。
  • 示例：
shell
more -dc /etc/services
1

第二章：wc 统计命令（⭐） ​

2.1 概述 ​

  • 命令：
shell
wc [-l] 文件
1
  • 对应的英文单词：word count，单词数。
  • 功能：统计文件的行数、单词数、字节数。
  • 选项：
    • -l，--line：统计行数。

注意⚠️：未来主要用于统计行数，包括：文件的行数或命令过滤结果的行数。

2.2 案例 ​

  • 示例：统计 /etc/services 文件的行数、单词数、字节数
shell
wc /etc/services
1

  • 示例：统计 /etc/services 文件的行数
shell
wc -l /etc/services
1

  • 示例：显示系统用户登录的错误信息
shell
# grep 是一种强大的文件搜索工具，可以将匹配的行打印出来
grep 'Failed password' /var/log/secure
1
2

  • 示例：统计系统用户登录的错误次数
shell

grep 'Failed password' /var/log/secure | wc -l
1
2

第三章：查询命令位置 ​

3.1 概述 ​

  • 在实际工作中，我们通常会使用 which 命令或 whereis 命令来显示命令所在位置的绝对路径。

3.2 which ​

  • 命令：
shell
which xxx ...
1

  • 功能：显示命令的所在位置的绝对路径。

  • 示例：查询 cd 命令的绝对路径

shell
which cd
1

  • 示例：查询 pwd 命令的绝对路径
shell
which pwd
1

  • 示例：查询 mkdir、touch、ls、mv 命令的绝对路径
shell
which mkdir touch ls mv
1

  • 示例：查询 ssh 命令的绝对路径
shell
which ssh
1

  • 示例：查询 java 命令的绝对路径
shell
which java
1

3.3 whereis ​

  • 命令：
shell
whereis xxx ...
1

  • 功能：显示命令及其帮助文件的所在位置的绝对路径。

  • 示例：查询 ssh 命令及其帮助文件的绝对路径

shell
whereis ssh
1

  • 示例：查询 java 命令及其帮助文件的绝对路径
shell
whereis java
1

  • 示例：查询 cd、head、tail、less、more 命令及其帮助文件的绝对路径
shell
whereis cd head tail less more
1
